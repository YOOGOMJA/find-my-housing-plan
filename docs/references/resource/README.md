# Resource 문서 모음

이 폴더는 SH/LH 공공주택 정보 수집 자동화를 위해 조회/검토한 외부 리소스 정보를 모아두는 공간입니다.

## 1) 요약

- 목적: API/데이터 소스 후보를 기록하고, 구현 시 참고할 링크를 보관
- 범위: 공공데이터포털, 기관 공식 사이트, 수집 전략 참고 정보
- 운영 원칙: 정보 변경 가능성이 있으므로 주기적으로 링크/상태를 재검증
- 도메인 기준이나 용어 해석은 먼저 `docs/domain/`에서 확인하고, 이 폴더는 그 판단을 보조하는 외부 근거와 링크를 정리하는 용도로 사용한다.

## 2) 현재 정리된 문서

- [SH-LH-API-리소스-정리.md](./SH-LH-API-리소스-정리.md)

## 3) 읽는 순서

1. `SH-LH-API-리소스-정리.md`의 `요약`과 `실행/적용 방법`부터 확인
2. 필요한 경우에만 API별 `세부 스펙/예외`와 `참고 링크` 확인

## 4) 주의

- 이 폴더는 1차 도메인 기준 문서가 아니다.
- 도메인 해석, 판정 규칙, 용어 정의가 필요하면 먼저 `docs/domain/` 문서를 확인한다.
